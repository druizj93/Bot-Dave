# Telegram AI Bot

Un bot de Telegram impulsado por IA con módulos de trading, YouTube y más.

## Características
- Acceso restringido a tu cuenta de Telegram
- Memoria persistente con SQLite
- Router inteligente para Gemini, DeepSeek, Mistral
- Módulo de Trading
- Módulo de YouTube
- Sistema de notificaciones
- Preparado para OpenAI

## Instrucciones

1. Clona o descarga el ZIP
2. `pip install -r requirements.txt`
3. Copia `.env.example` a `.env` y añade tus claves
4. Ejecuta `python bot.py`
5. Para producción: Conecta a Render
